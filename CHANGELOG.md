## [1.0.0] — 2025-07-01

### Added
- Initial portfolio release
- Hero section with typing animation, stats, and CTA buttons
- About section with skill bars and animated reveals
- Services grid (6 service cards)
- Portfolio section with category filter
- Freelance CTA section with enquiry modal
- Testimonials section
- Contact form with EmailJS integration
- Footer with social links and screen size previewer
- WhatsApp floating button (desktop → WhatsApp Web, mobile → WhatsApp app)
- Chat widget with lead capture (name, email, phone → sends to inbox)
- Screen size preview modal (Desktop / Laptop / Tablet / Mobile)
- Responsive design — fully mobile-friendly
- Scroll-triggered animations (IntersectionObserver)
- Toast notifications
- Separate About and Portfolio sub-pages
- README with setup and version control guide
